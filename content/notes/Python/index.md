## [Rye](https://rye-up.com/)
- MacOS Ventura 13.4.

### 導入

手順通りやるだけだが手もとでエラーが出たのでメモ．

```
curl: (6) Could not resolve host: rye-up.com
```

DNS を `8.8.8.8` と `8.8.4.4` にしたら解決した．

zsh にパスも通しておく．

```
echo 'source "$HOME/.rye/env"' >> ~/.zshrc
```
